◆ VCS_rp_drawto()

void VCS_rp_drawto ( VCS_renderpanel_t * source,
VCS_renderpanel_t * dest,
int x,
int y,
int transparent )
related

Draws a VCS_renderpanel_t onto another VCS_renderpanel_t with optional transparency. Useful for windows, widgets, etc.

Parameters
[in]sourceThe renderpanel to use as a drawing source.
[out]destThe renderpanel which will be drawn on.
[in]xThe x position of dest on which to place the top-left corner of source, where far left is 0.
[in]yThe y position of dest on which to place the top-left corner of source, where top is 0.
[in]transparentA value of 1 will cause all space characters in src to not replace characters in dest. A value of 0 disables this.
